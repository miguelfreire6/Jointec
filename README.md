# Jointec

Modern React + Vite landing page for an industrial company focused on wood recycling and pallet machinery.

## Stack

- React
- Vite
- Tailwind CSS

## Development

1. Install dependencies with `npm install`
2. Start the dev server with `npm run dev`
3. Build with `npm run build`

## Notes

This repository was scaffolded manually in an environment where Node.js was unavailable, so dependencies were not installed and the project was not executed locally here.
